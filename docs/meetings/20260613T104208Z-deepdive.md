# AgenTeam Deepdive: context-aware-music-recommender
Date: 2026-06-13 10:42

## Health: NO ACTIVE RUN

## External Signals (Researcher)

- Spotify live API dependency is now a project risk. The project should avoid live Audio Features, Audio Analysis, and Recommendations endpoints and use offline datasets plus reproducible snapshots.
- Open/public metadata and attributed chart snapshots are safer for coursework than scraping proprietary music platforms. Store source URLs and retrieval dates.
- Lyrics should be treated carefully: do not redistribute copyrighted lyrics unless the source explicitly permits it. Prefer licensed datasets or derived NLP features.
- Wikipedia Billboard scraping should be polite and attributed. Use a clear user agent, avoid bulk hammering, and document Wikimedia terms and robots/usage checks.
- Scenario-based recommendation is a strong fit for the teacher feedback because it evaluates suitability for explicit contexts instead of only surface metadata.
- Evaluation should compare baseline popularity/content similarity against scenario-aware ranking and include top-k, diversity, novelty, explainability, and scenario-fit metrics.
